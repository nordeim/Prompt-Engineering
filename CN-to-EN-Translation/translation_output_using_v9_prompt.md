## 📑 Content Overview (Chapters 1–10)

The book is built around the core formula **Agent = LLM + Context + Tools**, with ten chapters progressing layer by layer:

| Ch. | Topic | One-line takeaway | Text | Code |
| :--: | --- | --- | :--: | :--: |
| 1 | 🚀 **Agent Fundamentals** | The "model-as-Agent" paradigm + **Agent = LLM + Context + Tools**; Harness engineering is the real competitive edge | [Read](book/chapter1.md) | [4](chapter1/README.md) |
| 2 | 🎯 **Context Engineering** | Context sets the capability ceiling: KV Cache, prompt engineering, Agent Skills, context compression | [Read](book/chapter2.md) | [9](chapter2/README.md) |
| 3 | 📚 **User Memory & Knowledge Base** | Remember users across sessions, connect external knowledge: user memory, RAG, structured indexes, knowledge graphs | [Read](book/chapter3.md) | [13](chapter3/README.md) |
| 4 | 🛠️ **Tools** | Tools are the Agent's hands: MCP protocol, perception / execution / collaboration tool categories, event-driven async Agents | [Read](book/chapter4.md) | [6](chapter4/README.md) |
| 5 | 💻 **Coding Agent & Code Generation** | Code is "the tool that creates new tools" — a full panorama of production-grade Coding Agents | [Read](book/chapter5.md) | [12](chapter5/README.md) |
| 6 | 🎯 **Agent Evaluation** | Turn performance into comparable signals: evaluation environments, metrics, statistical significance, evaluation-driven model selection | [Read](book/chapter6.md) | [10](chapter6/README.md) |
| 7 | 🧠 **Model Post-Training** | Pre-training / SFT / RL in three stages: when to choose SFT vs. RL, internalizing tool calling, sample efficiency | [Read](book/chapter7.md) | [14](chapter7/README.md) |
| 8 | 🔄 **Agent Self-Evolution** | Grow without changing weights: learning from experience, active tool discovery, from user to creator | [Read](book/chapter8.md) | [7](chapter8/README.md) |
| 9 | 🎙️ **Multimodal & Real-Time Interaction** | Extend beyond text to speech, GUI, and the physical world: three speech paradigms, Computer Use, robotics | [Read](book/chapter9.md) | [7](chapter9/README.md) |
| 10 | 🤝 **Multi-Agent Collaboration** | Collective intelligence surpasses the individual: collaboration frameworks, shared / isolated context, emergent "Agent societies" | [Read](book/chapter10.md) | [6](chapter10/README.md) |

> 💡 **Read** = read the chapter text directly on GitHub (Markdown); **N** = number of companion projects in that chapter — click to view the code. See each chapter's README for project-type legends (✅ runnable / 📖 reproduction / 🚧 design doc).

## 🤝 Contributing

This book and its companion code are fully open source. Community contributions via Pull Request are warmly welcomed:

| Type | Description |
| --- | --- |
| 📝 **Book content improvements** | Errata, additions, clearer phrasing, or new cutting-edge developments (source text in `book/chapter*.md`) |
| 🐛 **Code improvements & bug fixes** | Make companion projects more robust, easier to use, and closer to production practice |
| 🧪 **New hands-on projects** | Supplement or replace an experiment with a better implementation, or contribute an entirely new example project |
| 🎨 **Figure design improvements** | Make the charts in `book/images/` clearer and more visually appealing (figures are generated by `book/gen_*_figs.py`) |
| 🌐 **New language translations** | Translations into additional languages are welcome — refer to the organization of Traditional Chinese (`book-zhtw/`), English (`book-en/`), Tamil (`book-ta/`), and Vietnamese (`book-vi/`) |

Before submitting, we recommend running the relevant experiments yourself to confirm reproducibility; you are also welcome to open an issue first to discuss ideas.
