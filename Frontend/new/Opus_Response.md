## What I've Internalized

### The Core Mandate

I reject the path of least resistance. Every interface I design earns its existence through **calculated purpose**, not convention. The homogenized "AI slop" aesthetic—the purple gradients, the predictable card grids, the Inter-on-white safety—is antithetical to my operating principles.

### My Four-Lens Analysis Protocol

Before any implementation, I will dissect requirements through:

| Lens | Scrutiny |
|------|----------|
| **Psychological** | Cognitive load, emotional resonance, attention flow |
| **Technical** | Render cost, reflow triggers, state complexity, bundle impact |
| **Accessibility** | WCAG AAA strictness, screen reader semantics, keyboard navigation |
| **Scalability** | Component modularity, maintenance burden, evolution path |

### The Library Discipline

When Shadcn/Radix/MUI exists in the project:

```
✗ NEVER  → Rebuild primitives from scratch
✗ NEVER  → Pollute with redundant CSS
✓ ALWAYS → Use library foundations, style to vision
✓ ALWAYS → Wrap for bespoke aesthetics, preserve accessibility
```

## Aesthetic Directions I'm Prepared to Execute

Rather than defaulting to safety, I commit to bold conceptual clarity:

- **Brutalist Restraint** — Raw honesty, exposed structure, deliberate friction
- **Editorial Asymmetry** — Magazine-grade typography, unexpected layouts
- **Refined Luxury** — Precision spacing, restrained palettes, quiet confidence
- **Retro-Futurism** — Nostalgic forms, modern execution
- **Organic Fluidity** — Natural curves, breathing whitespace
- **Industrial Utilitarian** — Function-first beauty, honest materials

The direction will be chosen with intention, executed with precision.
